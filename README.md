# US株価可視化ツール

## 概要

stream_practice1.py：streamlitでできることを個人の練習用に試しています。

stream_practice2.py：practice3のアプリ化に向けてデータ取得、グラフの表示を行います。

stream_practice3.py：米国株（Apple / Google / Microsoft）の過去の株価データを取得し、可視化するWebアプリです。
Streamlitを使用して簡単に操作でき、表示期間や株価レンジを自由に調整できます。

---

## 主な機能

* 指定日数分の株価データを取得
* 複数企業の株価を同時に比較表示
* 表示する企業を選択可能（マルチセレクト）
* 株価の表示範囲（y軸）を調整可能
* 折れ線グラフによる視覚的な分析

---

## 使用技術

* Python
* Streamlit（UI）
* yfinance（株価データ取得）
* pandas（データ処理）
* Altair（グラフ描画）

---

## 必要ライブラリのインストール

```bash
pip install streamlit yfinance pandas altair
```

---

## 使い方

### サイドバー操作

* **表示日数の選択**
  スライダーで過去何日分のデータを表示するか指定できます。

* **株価の範囲指定**
  グラフのY軸（株価）の表示範囲を調整できます。

---

### メイン画面

* **会社名の選択**
  表示したい企業を複数選択できます。

* **株価データ表示**

  * 表形式で株価データを確認
  * 折れ線グラフで推移を可視化

---

## データ仕様

* データ取得元：Yahoo Finance
* 使用データ：終値（Close）

---

## 注意点

* インターネット接続が必要です（株価取得のため）
* 取得できるデータは市場状況に依存します
* 表示日数が多い場合、処理が遅くなる可能性があります
